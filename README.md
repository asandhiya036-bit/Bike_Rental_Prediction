
# 🚴‍♂️ Bike Rental Demand Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting bike rental demand based on environmental and seasonal factors. Accurate demand prediction helps bike-sharing companies optimize inventory distribution, reduce shortages, and improve customer satisfaction.

---

## 🎯 Problem Statement

Bike-sharing systems generate large amounts of data related to rentals. The goal of this project is to build a machine learning model that predicts the number of bikes rented based on features such as weather conditions, time, and season.

---

## 📂 Dataset

The dataset includes historical bike rental data with the following key features:

* 🌡 Temperature
* 💧 Humidity
* 🌦 Weather Conditions
* 📅 Season
* 🕒 Hour / Day
* 🏢 Working Day / Holiday

---

## ⚙️ Project Workflow

1. **Data Cleaning**

   * Handled missing values
   * Removed duplicates
   * Converted categorical variables

2. **Exploratory Data Analysis (EDA)**

   * Analyzed rental trends across seasons and time
   * Visualized relationships between features
   * Identified key factors affecting demand

3. **Feature Engineering**

   * Encoded categorical variables
   * Created meaningful features for better prediction

4. **Model Building**

   * Linear Regression
   * Decision Tree Regressor
   * Random Forest Regressor

5. **Model Evaluation**

   * Compared models using evaluation metrics
